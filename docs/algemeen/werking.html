Algemeen: werking

Via de API voert u geautomatiseerd opdrachten uit vanuit uw eigen systeem. U stelt het gewenste verzoek samen en verzend deze naar MijnDomeinReseller. Vervolgens ontvangt u antwoord op uw verzoek.

Een opdracht samenstellen

Opdrachten worden via een GET request aan de API verzonden. Zie het volgende voorbeeld voor een whois opdracht:

command=whois
user=username
pass=password
type=uitgebreid
domein=mijndomeinreseller.nl

Deze waarden worden als volgt verzonden naar de API:
https://manager.mijndomeinreseller.nl/api/index.php?command=whois&user=username&pass=password&type=uitgebreid&domein=mijndomeinreseller.nl

Het antwoord ziet er als volgt uit:

domein=mijndomeinreseller.nl
result=whois data
errcount=0
done=true

Het antwoord kunt u in uw eigen systeem verwerken.

PHP class

Voor een eenvoudige werking is een PHP class beschikbaar. Deze PHP class kunt u in uw eigen PHP script verwerken. De voorbeelden in deze html handleiding zijn gebaseerd op de PHP class.

Test omgeving

U kunt gebruik maken van de test omgeving om de aansluiting van uw systeem op de API te testen. Bij gebruik van de test omgeving worden mutaties aan domeinnamen niet doorgevoerd. U kunt de test omgeving aan/uit zetten in de domeinmanager.
