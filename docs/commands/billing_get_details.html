Command: billing_get_details

Omschrijving

Vraag facturatie op van domeinnamen (enkel voor postpaid betalen)

Invoer parameters

Parameter Waarde Verplicht
 
user Username API Ja
pass Wachtwoord API Ja
authtype Wachtwoord wordt verzonden in plaintext/md5
waarden: plain/md5		 Ja
 
domein Domeinnaam nee
tld TLD (extensie) Ja
 
jaar Toon gefacturerde domeinnamen van opgegeven jaar en maand Nee, alleen als maand is opgegeven
maand Toon gefacturerde domeinnamen van opgegeven jaar en maand 01 t/m 12 Nee, alleen als jaar is opgegeven
 


Uitvoer parameters

Parameter Waarde
billing_details_count Numerieke waarden van het aantal gefactureerde domeinen
 
datum[n] Datum van facturatie
prijs[n] Prijs van facturatie
domein[n] Gefactureerde domeinnaam
tld[n] Extensie van domeinnaam
omschrijving[n] Volledige omschrijving
n wordt vervangen door een opeenlopend cijfer, beginnend bij 0, eindigt bij billing_details_count
 
errcount 0
done true
