Command: contact_list

Omschrijving

Toon een overzicht van de contacts uit het account

Invoer parameters

Parameter Waarde Verplicht
 
user Username API Ja
pass Wachtwoord API Ja
authtype Wachtwoord wordt verzonden in plaintext/md5
waarden: plain/md5		 Ja
 
sort Sorteren contacts, waarden: bedrijfsnaam/voornaam/tussenvoegsel/achternaam/email/tel Nee
 
order Sorteer contact oplopend/aflopend, waarden: 0 voor oplopend, 1 voor aflopend Nee


Uitvoer parameters

Parameter Waarde
contactcount Numerieke waarden van het aantal contacts
 
contact_id[n] ContactID
contact_bedrijfsnaam[n] Bedrijfsnaam van contact
contact_voorletter[n] Voorletter van contact
contact_tussenvoegsel[n] Tussenvoegsel van contact
contact_achternaam[n] Achternaam van contact
contact_straat[n] Straat van contact
contact_huisnr[n] Huisnummer van contact
contact_postcode[n] Postcode van contact
contact_plaats[n] Plaats van contact
contact_land[n] Landcode (bijlage 3) van contact
contact_email[n] Emailadres van contact
contact_tel[n] Telefoonnummer van contact
n wordt vervangen door een opeenlopend cijfer, beginnend bij 0, eindigt bij contactcount
