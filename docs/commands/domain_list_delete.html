Command: domain_list_delete

Omschrijving

Toon een overzicht van de domeinnamen welke in de afgelopen 30 dagen uit het account zijn wegverhuisd

Invoer parameters

Parameter Waarde Verplicht
 
user Username API Ja
pass Wachtwoord API Ja
authtype Wachtwoord wordt verzonden in plaintext/md5
waarden: plain/md5		 Ja


Uitvoer parameters

Parameter Waarde
domain_list_delete_count Nummerieke waarde van het aantal domeinnamen
 
domein[n] Domeinnaam
tld[n] TLD van domeinnaam
datum[n] Datum van wegverhuizing
type[n] Type verwijdering, toont enkel "verhuizing"
n wordt vervangen door een opeenlopend cijfer, beginnend bij 0, eindigt bij domain_list_delete_count
 
errcount 0
done true
