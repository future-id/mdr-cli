Command: domain_modify_ns

Omschrijving

Wijzig de nameservers van een domeinnaam

Invoer parameters

Parameter Waarde Verplicht
 
user Username API Ja
pass Wachtwoord API Ja
authtype Wachtwoord wordt verzonden in plaintext/md5
waarden: plain/md5		 Ja
 
domein Domeinnaam Ja
tld TLD (extensie) van domeinnaam Ja
 
gebruik_dns Gebruik de nameservers van MijnDomeinReseller, waarden: true/false Nee
dns_template Naam van DNS template Nee, alleen mogelijk i.c.m. gebruik_dns=true
 
ns_id nameserverID van nameserverset Ja, als ns1, ns1_ip, ns2 en ns2_ip niet is opgegeven en gebruik_dns = false
ns1 Nameserver 1 Ja, als ns_id niet is opgegeven en gebruik_dns = false
ns1_ip Ipadres van Nameserver 1 Ja, Als ns1 is nsX.domein.tld
ns2 Nameserver 2 Ja, als ns_id niet is opgegeven en gebruik_dns = false
ns2_ip Ipadres van Nameserver 2 Ja, Als ns2 is nsX.domein.tld
ns3 Nameserver 3 Nee
ns3_ip Ipadres van Nameserver 3 Ja, Als ns3 is opgegeven en ns3 is nsX.domein.tld

Parameter Waarde
errcount 0
done true
