Command: domain_restore

Omschrijving

Heractiveer een domeinnaam uit quarantaine
Beschikbaar voor .NL, .BE en .EU domeinnamen

Invoer parameters

Parameter Waarde Verplicht
 
user Username API Ja
pass Wachtwoord API Ja
authtype Wachtwoord wordt verzonden in plaintext/md5
waarden: plain/md5		 Ja
 
domein Domeinnaam Ja
tld TLD (extensie) van domeinnaam Ja


Uitvoer parameters

Parameter Waarde
errcount 0
done true


Opmerkingen

1. Aan het heractiveren van een domeinnaam zijn hogere kosten verbonden, bekijk de kosten in uw account.
